How the Arcade Works

Every arcade game feeds into the same progression loop. Here is what happens each time you play:

  1. Pick any arcade game from the Arcade.
  2. Play a run — your session builds a score in real time.
  3. Submit your score at game over — accepted scores go on record.
  4. The score gives you Arcade XP — XP is the progression signal across the arcade (requires Telegram link to persist server-side).
  5. Arcade XP updates your account — synced when Telegram is linked.

Your Arcade XP counts toward:

  • Leaderboard position — score rank on the Leaderboard (score only, XP does not affect rank)
  • Block Topia entry gate — 50 Arcade XP unlocks Block Topia (requires Telegram sync)
  • Faction XP — joining a faction earns separate Faction XP that tracks your alignment level

Note: Daily Missions, Seasons, and Faction war standings are locally scaffolded. They track progress client-side but are not yet fully server-backed. Do not rely on them for persistent cross-device progress.

Core loop: Arcade → score → accepted run → Telegram sync → Arcade XP → 50 XP unlocks Block Topia.

Game Types

The arcade contains multiple game types. Each type rewards different skills and playstyles. Different game types can also receive different Cross-Game Modifiers.

🚀 Shooter Games

Examples: Invaders 3008, Asteroid Fork

Best for: reaction speed, survival, boss damage

🌀 Maze Games

Example: Pac-Chain

Best for: routing, risk/reward, streak control

🐍 Snake Games

Example: SnakeRun 3008

Best for: combo chains, survival, pickups

🧱 Breakout Games

Example: Breakout Bullrun

Best for: physics skill, combos, pressure control

🧩 Puzzle / Block Games

Examples: Tetris Block Topia, Crystal Quest

Best for: planning, endurance, clean scoring

Different game types can receive different Cross-Game Modifiers — so picking the right game type during an active modifier can make a real difference to your run.

Cross-Game Modifiers Partially wired · local only

Cross-Game Modifiers are temporary arcade-wide bonuses. The modifier definitions and compatibility tags are wired into each game. Modifier state is stored locally — it is not server-backed or cross-device.

📈 Score Surge

More score gained during the run.

🛡️ Shielded Start

Start with extra protection where the game supports it.

🐢 Slow Chaos

Chaos events and pressure build more slowly.

🌟 Golden Chance

Rare rewards appear more often during the run.

💀 Boss Hunter

Improved boss damage where bosses exist in the game.

Important to know:

  • Not every Cross-Game Modifier affects every game
  • Modifiers only apply to compatible game types
  • No modifier changes the core Arcade XP formula directly
  • Modifier persistence is stored locally in this browser and can persist between sessions, but it is not cross-device

Daily Missions Scaffolded · local only

Daily Missions are defined and tracked locally in your browser. Mission progress resets daily and is not synced server-side or cross-device.

Missions may ask you to:

  • Survive longer in a run
  • Complete multiple runs
  • Trigger chaos events
  • Collect rare rewards
  • Contribute to a Faction war effort
  • Hit score targets in specific game types
  • Daily Missions refresh — new objectives appear regularly (local rotation)
  • Mission progress is browser-only — not backed by the server at this stage

Season Competitions Coming next

Season competitions are planned but not yet fully live. The leaderboard currently tracks all-time scores. Seasonal windows and standing resets are not yet active.

  • Scores and Arcade XP will build seasonal standing once seasons go live
  • Every Season will give players a fresh competition window
  • Seasons reset the competition energy so standings remain contested

No season rewards are promised at this stage.

Factions / Clans Live · gameplay effects active

Players can align with a Faction. Faction effects are wired into compatible arcade games — they adjust score multipliers, chaos rates, shield counts, and combo bonuses inside those games.

Faction XP is separate from Arcade XP. Faction XP tracks your alignment level and faction standing only. It does not count toward the Block Topia gate.

Faction war standings (who is winning the season war) are currently tracked locally in your browser. Cross-player faction war results are not yet server-backed.

💎 Diamond Hands

  • Endurance-focused playstyle
  • Survival bonus after 30 s alive
  • Lower chaos pressure during runs

🛡️ HODL Warriors

  • Defense-focused playstyle
  • Extra shield and protection where supported
  • Built for survival-heavy strategies

🎨 GraffPUNKS

  • Chaos and combo-focused playstyle
  • Higher chaos pressure during runs
  • Stronger combo identity and multipliers

Key facts about Factions:

  • Faction gameplay effects are active in: Invaders 3008, Pac-Chain, Tetris Block Topia, Breakout Bullrun, Asteroid Fork, SnakeRun 3008
  • Faction XP is earned via /faction/earn — separate from Arcade XP
  • Faction war standings are local-only at this stage
Choose a Faction that matches how you want to play. Faction effects change conditions inside your run right now.

XP: Your Progress Signal

XP is not just a number. It is proof that you showed up. The arcade tracks four separate signals — they are not interchangeable.

  • Score — determines leaderboard rank only. Does not affect any XP.
  • Arcade XP — earned from accepted synced runs. Gates Block Topia entry. Requires Telegram link to persist.
  • Faction XP — earned by joining and playing under a faction. Tracks alignment level. Does not count toward Block Topia gate.
  • Block Topia XP — in-game progression inside Block Topia itself. Separate from all of the above.
  • Arcade XP helps unlock deeper systems
  • Arcade XP matters more when your Telegram identity is linked
  • 50 Arcade XP is required to enter Block Topia

Telegram Sync: Save Your Position

If you want progression to matter, link Telegram. This connects your player identity to the live system.

  1. Open @WIKICOMSBOT.
  2. Run /gkstart.
  3. Run /gklink.
  4. Click the link from the bot.
  5. Return to the site and continue playing.

If sync goes red or expires, run /gklink again.

Quick Fixes

I played but got no XP

Score and XP are separate. XP requires accepted score conversion and Telegram sync.

I cannot enter Block Topia

You probably need more XP or need to relink Telegram. 50 Arcade XP is required.

Why does it say local only?

You are not synced. Local runs stay in the browser and do not act like full progression.

Sync is red

Run /gklink again and open the new link from the bot.
