Blockchain Technology

✅ Verified📅 Last updated: 2024-01📂 Technology
📋 Contents

A blockchain is a type of distributed ledger technology (DLT) in which data is stored in blocks that are cryptographically linked in a chronological chain. Once a block is added to the chain, it is extremely difficult to alter — making blockchain a powerful tool for recording transactions and data that must be trusted by all parties.

How It Works

Blocks

Each block contains: a header (including timestamp, nonce, and the hash of the previous block), a body (the actual transaction data), and a Merkle root — a cryptographic summary of all transactions in the block. Linking each block to the previous via its hash creates the "chain" — altering any past block would invalidate all subsequent blocks.

Consensus Mechanisms

Because there is no central authority, blockchains rely on consensus mechanisms to ensure all nodes agree on the state of the ledger:

  • Proof of Work (PoW) — Nodes compete to solve cryptographic puzzles (Bitcoin)
  • Proof of Stake (PoS) — Validators are chosen based on staked tokens (Ethereum post-Merge)
  • Delegated PoS — Token holders vote for validators (EOS, Tron)
  • Proof of History — Cryptographic clock for ordering events (Solana)

Decentralisation

Blockchain's most important property is decentralisation — there is no single point of control or failure. Copies of the entire blockchain exist on thousands of nodes worldwide. For an attacker to modify historical data on Bitcoin, they would need to control over 51% of the total mining hash power — making it economically infeasible.

The Blockchain Trilemma

Coined by Vitalik Buterin, the blockchain trilemma states that a blockchain can only optimise for two of three properties at once:

  • 🔒 Security — Resistance to attacks
  • ⚡ Scalability — Transaction throughput
  • 🌐 Decentralisation — Distributed control

Bitcoin prioritises security and decentralisation at the cost of scalability. Layer 2 solutions (Lightning Network, Rollups) attempt to resolve this without compromising the base layer.
