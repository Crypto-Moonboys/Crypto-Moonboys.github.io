⚙️ How Monster Mode Works

  • Race the full HexGL circuit loaded directly from the official CDN.
  • Click Start inside the HexGL game frame to begin racing, then click ⏱ Begin Tracked Run to start the wrapper timer.
  • When your run ends, click Submit Run to log your time.
  • Score = 500 000 − (time_seconds × 1000). Faster = higher score.
  • Scores above 0 qualify for the seasonal leaderboard and XP pool.
  • Telegram sync required for competitive ranking — guest runs are local only.
