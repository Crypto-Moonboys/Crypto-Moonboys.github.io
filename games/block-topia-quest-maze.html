Block Topia Quest Maze

A Crystal Maze-style companion RPG built on top of the wiki itself. Your guide sends you into live lore zones, points you at the right page, and expects you to come back with the answer. The game keeps two anti-repeat 90-day memory files per player, prepares for R2-backed progression, and is designed to ingest fresh wiki updates into future quest banks automatically.

90-Day Infinity Model
Each user carries two remembered 90-day files. The active file stores the current season. The shadow file stores the previous season. A future scheduler excludes exact quest ids, normalized answers, clue families, and source-page repeats from both files before assigning a new daily quest. Seasonal public scores wipe every 90 days. The masters board keeps the all-time top ten only.
Automated Systems Included
Companion guidance, daily quest scheduling, two-slot anti-repeat memory, narration hooks, locale-ready shell, accessibility preferences, wiki-update quest harvesting notes, and R2/API storage contracts. This page runs as a front-end demo now, but it is shaped so a Worker can move state into Cloudflare R2 without ripping out the game logic.
