Moonboys hero asset pack

Files:
- scene_full.png: original full image converted to PNG
- section_top.png / section_middle.png / section_bottom.png: the full image split into 3 horizontal sections that reconstruct the whole scene
- animation_overlay_top.png: transparent overlay for sky / clouds / moon / stars
- animation_overlay_middle.png: transparent overlay for skyline neon lights
- animation_overlay_bottom.png: transparent overlay for bridge / water reflections

Intended use:
1. Stack each overlay directly above its matching base section.
2. Animate top overlay with slow sideways drift (cloud motion).
3. Animate middle overlay with gentle opacity flicker (window / neon flicker).
4. Animate bottom overlay with gentle opacity shimmer (water / bridge reflection flicker).

All section images keep the original full width, so no top/middle/bottom content is missing.
