🤖 How It Works

The Crypto Moonboys Wiki uses an automated publishing pipeline:

  • ⚡️ brain — knowledge store and agent orchestrator
  • ⚡️ memory — validated entity data and lore facts
  • ⚡️ publisher — converts memory into clean HTML pages
  • ⚡️ wiki — this static site (GitHub Pages)
  • ⚡️ telegram — community notifications

📋 Publishing Rules

All published articles follow strict rules defined in brain-rules.md:

  • ⚡️ Rule 1 — Every new page must appear in the ⚔️ HODL Wars Lore$ sidebar
  • ⚡️ Rule 2 — Every sidebar entry must have a fitting emoji
  • ⚡️ Rule 3 — Emoji-first titles and headers
  • ⚡️ Rule 5 — Every page must cross-link to at least 4 other pages
  • ⚡️ Rule 6 — No lorem ipsum, no placeholder content — real, quality writing only
  • ⚡️ Rule 7 — sitemap.xml must be updated with every new page
  • ⚡️ Rule 8 — No legacy sam- URL prefixes — descriptive slugs only

Full rules: brain-rules.md on GitHub →
