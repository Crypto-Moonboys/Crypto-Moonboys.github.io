# Copilot Instructions — Crypto Moonboys

## Repository purpose
This is the live website repository for Crypto Moonboys. It hosts the static
site, the arcade games, and shared frontend assets. Keep changes focused and
minimal.

## Anti-drift rules (enforced by CI)
See `AGENT_ENFORCEMENT.md` for the full list of enforced structural rules.
The script `scripts/anti-drift-check.mjs` runs on every PR. Do not bypass it.

Key rules at a glance:

- **Required root files** (`README.md`, `robots.txt`, `.nojekyll`, `index.html`,
  `CNAME`) must not be deleted or moved.
- **All active arcade games** must remain under `games/`.
- **Block Topia** must stay in its clean minimal state — do not re-introduce
  `world/`, `ui/`, `economy/`, or `duel/` directories, or any
  `PressureProtocol` / `street-signal` / solo-mode references.
- **HexGL** is deprecated. Do not re-enable score submission in
  `hexgl-monster-max/bootstrap.js`.
- `robots.txt` must remain at the repository root.
- Active game `index.html` files must not reference forbidden scripts.

## Workflow rules
- Open **one PR per task**. Do not merge automatically.
- The anti-drift workflow must pass before any PR is merged.
- Do not delete or disable `.github/workflows/anti-drift-check.yml`.

## Coding conventions
- Scripts in `scripts/` use `.mjs` for Node ESM when they need Node APIs.
- Game pages use `type="module"` ES modules — avoid CommonJS patterns in
  browser-facing code.
- No new build tools or bundlers. The site is served as static files.
- Load `/js/api-config.js` before any bootstrap script that uses
  `MOONBOYS_API.BASE_URL`.

## Arcade XP source of truth
All active arcade games participate in shared Arcade XP progression after Telegram sync. Invaders 3008 is the current season anchor game but is **not** the sole XP source. Score submission and ArcadeSync are wired into all active arcade games. Do not disable XP sync from any active game without a deliberate product decision documented in the PR.

## Do not commit
- Secrets, API keys, or tokens.
- Binary assets larger than 500 KB without explicit approval.
- Auto-generated files without regenerating them through the documented script.
